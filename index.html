Setup Instructions to use NoKYCTranslate with the Damus Nostr App
Step-by-Step Instructions

  1. Tap your profile pic in the top left
  2. Tap Settings
  3. Tap Translations
    • Service: NoKYCTranslate.com
    • API Key: (Enter your API Key from above, copy and paste)
    • Automatically translate notes: If you turn this on then notes will be translated as you scroll and the note will say "Translated from ...." and then underneath will be the translation. If you keep this feature off, then you can manually translate notes by clicking the "..." at the top of any note and selecting "Translate".
  4. You should be good to go and start to see apps being translated!
